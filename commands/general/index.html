    Commands Tutorials API
!abilities <term> Shows abilities of a following: <speedster | viber | kryptonian>
!crossovers Displays a list of all Arrowverse crossover episodes in their respective watching order
!ep <show> <NxNN|SNENN> ~ N -> number Fetches episode info
!next <show> <NxNN/SNNENN> ~ N -> number Fetches next episode in watching order
!nxeps Displays a countdown to the next airing Arrowverse episodes
!soundtracks Displays all soundtracks
!sp <name/alter ego> eN (earth number) Fetches Speedster info
!subs <lang> <show> <NxNN/SNNENN> ~ N -> number Searches opensubtitles.org for the specified episode
!suit <character> Displays newly revealed suits
!torrent <show> <NxNN/SNNENN> ~ N -> number Searches torrent providers for the specified episode
!wiki[stg|kr|lu|dc] <term> Searches the specified wiki for the given term | stg - Stargirl | kr - Krypton | lu - Lucifer | dc - DC |
