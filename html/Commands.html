Commands List

!abilities (term)
Shows abilities of a following: Speedster | Viber | Kryptonian>
!crossovers
Displays a list of all Arrowverse crossover episodes in their respective watching
!ep (show) (NxNN|SNENN) ~ N ->
Fetches episode info
!next (show) (NxNN/SNNENN) ~ N -> number
Fetches next episode in watching order
!nxeps
Displays a countdown to the next airing Arrowverse episodes
!soundtracks
Displays all soundtracks
!sp (name/alter ego) EN (earth number)
Fetches Speedster info
!subs (lang) (show) (NxNN/SNNENN) ~ N -> number
Searches opensubtitles.org for the specified episode
!suit (character)
Displays newly revealed suits
!torrent (show) (NxNN/SNNENN) ~ N -> number
Searches torrent providers for the specified episode
!wiki[stg|kr|lu|dc] (term)
Searches the specified wiki for the given term | stg - Stargirl | kr - Krypton | lu - Lucifer | dc - DC |
